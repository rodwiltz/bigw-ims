# Project Victory

## Mission

Build a simple, reliable inventory management system that enables Big W to grow beyond a single delivery location.

## Core Principles

- Scan instead of type.
- Every item has a unique identity.
- Inventory always has one current location.
- Every scan changes inventory state.
- The software should reduce mistakes, not create work.
- The system should be simple enough that a new employee can learn it in minutes.

## Long-Term Vision

Support multiple pickup locations, multiple employees, and complete inventory visibility while remaining affordable for a small business.
